Detailed Description

Represents a JSON value in a parser-library-neutral kind of way. It throws VoltDB-style exceptions when things are amiss and should be otherwise pretty simple to figure out how to use. See plannodes or expressions for examples.

It might require some fudging to move from rapidjson to jsoncpp or something, but WAY less fudging than it would take to edit every bit of code that uses this shim.
