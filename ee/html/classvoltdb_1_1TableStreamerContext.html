virtual ActivationReturnCode voltdb::TableStreamerContext::handleActivation ( TableStreamType  streamType  )  [inline, virtual]

Optional activation handler. Called after creating the context to see if activation is allowed. Return ACTIVATION_SUCCEEDED if (re)activation is allowed and succeeded. Return ACTIVATION_FAILED if (re)activation is allowed but failed. Return ACTIVATION_UNSUPPORTED if (re)activation is not supported for the stream type.

Reimplemented in voltdb::CopyOnWriteContext, voltdb::ElasticContext, and voltdb::ElasticIndexReadContext.

virtual bool voltdb::TableStreamerContext::handleDeactivation ( TableStreamType  streamType  )  [inline, virtual]

Optional deactivation handler. Called when the stream is shutting down. Return true to keep it around and listening to updates. (default=false)

Reimplemented in voltdb::ElasticContext, and voltdb::ElasticIndexReadContext.
