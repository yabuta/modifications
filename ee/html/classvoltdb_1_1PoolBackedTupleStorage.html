Detailed Description

Convenience class for Tuples that get their (inline) storage from a pool. The pool is specified on initial allocation and retained for later reallocations. The tuples can be used like normal tuples except for allocation/reallocation. The caller takes responsibility for consistently using the specialized methods below for that.
