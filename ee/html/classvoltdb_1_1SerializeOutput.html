Detailed Description

Abstract class for writing to memory buffers. Subclasses may optionally support resizing.

virtual void voltdb::SerializeOutput::expand ( size_t  minimum_desired  )  [protected, pure virtual]

Called when trying to write past the end of the buffer. Subclasses can optionally resize the buffer by calling initialize. If this function returns and size() < minimum_desired, the program will crash.

Parameters:
minimum_desired the minimum length the resized buffer needs to have.

Implemented in voltdb::ReferenceSerializeOutput, voltdb::FallbackSerializeOutput, and voltdb::CopySerializeOutput.
