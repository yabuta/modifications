Detailed Description

template<class K, class T, class H = boost::hash<K>, class EK = std::equal_to<K>, class ET = std::equal_to<T>>
struct voltdb::CompactingHashTable< K, T, H, EK, ET >::HashNodeSmall

HashNodeSmall is some kind of dark evil magic. It is the same as the HashNode, but missing the pointer that is only used for multimap indices at the end. Using casts, unique indexes will only use this type underneath, and thus save 8B per key.
