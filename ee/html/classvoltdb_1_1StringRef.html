Detailed Description

An object to use in lieu of raw char* pointers for strings which are not inlined into tuple storage. This provides a constant value to live in tuple storage while allowing the memory containing the actual string to be moved around as the result of compaction.

Pool *  dataPool 
) [static]

Create and return a new StringRef object which points to an allocated memory block of the requested size. The caller may provide an optional Pool from which the memory (and the memory for the StringRef object itself) will be allocated, intended for temporary strings. If no Pool object is provided, the StringRef and the string memory will be allocated out of the ThreadLocalPool.
