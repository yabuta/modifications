Debug logging functions for EE. Unlike the performance counters, these are just printf() turned on/off by VOLT_LOG_LEVEL compile option. The main concern here is not to add any overhead on runtime performance when the logging is turned off. Use VOLT_XXX_ENABLED macros defined here to eliminate all instructions in the final binary.
