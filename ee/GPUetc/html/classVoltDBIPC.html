char * VoltDBIPC::retrieveDependency ( int32_t  dependencyId,
size_t *  dependencySz 
)

Retrieve a dependency from Java via the IPC connection. This method returns null if there are no more dependency tables. Otherwise it returns a pointer to a buffer containing the dependency. The first four bytes of the buffer is an int32_t length prefix.

The returned allocated memory must be freed by the caller. Returns dependency size with out parameter.
