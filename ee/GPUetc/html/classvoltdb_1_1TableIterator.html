Detailed Description

Iterator for table which neglects deleted tuples. TableIterator is a small and copiable object. You can copy it, not passing a pointer of it.

This class should be a virtual interface or should be templated on the underlying table data iterator. Either change requires some updating of the iterators that are persvasively stack allocated...
