Detailed Description

An implementation of a LogProxy that forwards the log statements to the java org.voltdb.jni.EELoggers class using JNI.

void voltdb::JNILogProxy::setJNIEnv ( JNIEnv *  env  )  [inline]

Update the reference to the JNIEnv object that is used to invoke java logging methods. The env pointer must be updated EVERY time there is a transition from Java to native code where this log proxy might be used. The env pointer is unique to each thread and can change every time there is a transition from Java to C.
